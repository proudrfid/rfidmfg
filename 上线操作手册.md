## 第 1 步:注册域名 `rfidmfg.com` ⏱️10 分钟

- 去任一注册商:**Cloudflare Registrar**(最便宜、和后面部署同平台,推荐)、阿里云、腾讯云、Namecheap 均可。
- 搜索 `rfidmfg.com`,下单注册(约 ¥70–100/年)。
- 如果用 Cloudflare 部署,**强烈建议域名也放 Cloudflare**,后面绑定和 HTTPS 一键完成。

> ⚠️ 全站的网址、canonical、sitemap、邮箱都已按 `www.rfidmfg.com` / `@rfidmfg.com` 写好,所以域名必须是这个。

## 第 4 步:部署上线(Cloudflare Pages,免费) ⏱️15 分钟

**最简单的方式 — 直接上传整个文件夹:**

1. 注册/登录 [Cloudflare](https://dash.cloudflare.com) → 左侧 **Workers & Pages** → **Create** → **Pages** → **Upload assets**。
2. 项目名填 `rfidmfg`,把整个 `mindrfid-website` 文件夹**拖进去**上传 → Deploy。
3. 部署完会给你一个 `xxx.pages.dev` 临时网址,先打开自查一遍。
4. 进项目 → **Custom domains** → 添加 `www.rfidmfg.com` 和 `rfidmfg.com` → 按提示(域名在 Cloudflare 的话自动)→ **HTTPS 自动开启**。
5. 设置把 `rfidmfg.com` 跳转到 `www.rfidmfg.com`(或反过来,二选一,和 canonical 一致即可,我们用的是 `www`)。

> 其他可选主机:**GitHub Pages**(免费,需会用 Git)、**Vercel / Netlify**(拖拽部署,体验类似)。  
> 以后改了网站,重新跑一次 `node build-products.js && node build-articles.js`,再重新上传即可。

## 第 5 步:接询盘表单 + 装统计

### 5a. 把联系表单接到飞书多维表格 ⏱️20 分钟

现在表单是 `mailto` 兜底(会打开邮件客户端),**不会自动存表**。要自动收集线索:

1. ⚠️ **先重置飞书 App Secret**(之前明文发过):飞书开放平台 → 你的应用 → 凭证与基础信息 → 重置 App Secret。
2. 部署 `feishu-lead-worker.js` 到 **Cloudflare → Workers**(新建 Worker,把文件内容贴进去,Deploy)。
3. 在 Worker → Settings → Variables 配 5 个环境变量:
   - `FEISHU_APP_ID` — 你的 App ID(`cli_` 开头)
   - `FEISHU_APP_SECRET` — **刚重置的新** Secret
   - `APP_TOKEN` — 多维表格链接里 `/base/` 后那串
   - `TABLE_ID` — 链接里 `?table=` 后那串
   - `ALLOW_ORIGIN` — `https://www.rfidmfg.com`
4. 多维表格建一张"线索表",字段:`姓名 / 邮箱 / 产品 / 留言 / 来源 / 提交时间`。
5. 把 Worker 的网址,填进 `script.js` 顶部的 `FORM_ENDPOINT = ''`(引号里),重新部署网站。
6. 自己在网站上提交一条测试,确认多维表格里出现记录。

> 不想搭 Worker?也可以用 **Formspree**(把表单 action 指向它的网址即可),更简单但线索进的是 Formspree 后台而非飞书。要哪种我都能帮你配。

### 5b. 装 Google Analytics 4(看流量) ⏱️10 分钟

每页 `<head>` 里已经放好 GA4 代码,**默认是注释状态(不发数据)**。激活:

1. 建 GA4 媒体资源,拿到 `G-` 开头的衡量 ID。
2. 把全站的 `G-XXXXXXXXXX` 替换成你的 ID,并去掉那段注释的 `<!-- … -->` 外壳。
3. 这步我可以**一条命令帮你全站替换**——你把 ID 发我即可。

### 5c. 提交搜索引擎收录(必做,否则没自然流量) ⏱️15 分钟

- **Google Search Console**(search.google.com/search-console):验证 `rfidmfg.com` → 提交 `https://www.rfidmfg.com/sitemap.xml`。
- **Bing Webmaster Tools**:同样验证 + 提交 sitemap(Bing 也喂 ChatGPT 等 AI,别漏)。
- `robots.txt` 已放行所有主流 AI 爬虫(GPTBot、PerplexityBot、ClaudeBot 等)。
